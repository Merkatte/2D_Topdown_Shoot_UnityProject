# 2D Topdown Shooter

Unity로 제작한 2D 탑다운 슈팅 게임 프로젝트입니다.

## 프로젝트 소개
탑다운 시점의 슈팅 게임으로, 플레이어가 적을 처치하며 생존하는 게임입니다.

## 개발 환경
- **Engine**: Unity 6000.2.7f2
- **Language**: C#
- **Platform**: PC

## 현재 기능
- 탑다운 시점 캐릭터 이동
- 슈팅 시스템

## 제작 중
- 적 스탯 정보 적용
- 적 AI
- 유닛 라이프 사이클
- 공격 판정에 따른 결과 로직
